# MyTimeline-with-reactjs

This repo is using reactjs(typescript), react-chrono, and Contentful.

### Blog
https://dev.to/kojikanao/create-timeline-with-reactjs-and-contentful-4j7p

## How to run
```
$ git clone
$ cd MyTimeline-with-reactjs
$ yarn
$ yarn start
```
